## Installation

```
$ git clone git@github.com:BoisselNicolas/TicTacToeCPP.git
```

## Compilation


```
$ g++ main.cpp -o main
```
Le paramètre "-o" définit le nom qu'aura notre éxécutable.

## Usage 

Pour éxécuter le programme

```
$ ./main
```
